# Chuni-Eventer v0.6.3

## 更新内容

### 本版重点

- 新增 **PJSK 自助称号制作器**，可在本地快速生成并落库称号资源，减少手工拼 XML 的步骤。
- 新增 **单立绘自助裁剪功能**，从输入图到可用资源的处理链路更直接，降低立绘制作门槛。
- 优化启动阶段的初始化流程，**大幅提升启动速度**，降低首次打开等待时间。
- 解决部分 **弹窗卡死/界面无响应** 场景，提升编辑与切换页面时的稳定性。
- 修复 **CharaWorks 索引问题**，减少角色作品信息错位或无法正确命中的情况。

### 体验与稳定性

- 优化部分编辑链路的容错提示，减少异常时的中断感。
- 对资源索引与读取流程做小幅整理，提升维护性。

## 打包说明

- Windows 一键打包：

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\build_windows.ps1" -Version 0.6.3
```

- 详见 `packaging/BUILD_AND_DISTRIBUTION.md`。

## 升级提示

- 建议升级前备份当前 ACUS 与自定义资源目录。
- 若资源管理器图标显示未即时更新，可刷新或重启资源管理器后再查看。
