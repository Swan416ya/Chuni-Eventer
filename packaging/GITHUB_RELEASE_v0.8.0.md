# Chuni-Eventer v0.8.0

## 更新内容

### 新功能

- **段位随机选曲**：创建段位课时，每首歌可在三种模式间切换——固定选曲、按等级随机（Lv10~Lv15+）、按候选池随机（手动指定曲目列表）。支持混合模式（同一课内不同歌曲用不同模式），编辑现有随机段位时自动识别并回显。
- **资源压缩包快捷导入**：在「设置 - 资源导入」中导入他人分享的资源压缩包，自动检测与当前 ACUS 的 ID 冲突并重映射到空闲 ID，包内所有 XML 引用关系同步更新。导入前可预览包内全部资源及原 ID / 新 ID 对照表。
- **默认挡板贴图替换**：在「设置 - 实验性功能」中上传图片，自动生成 `ddsFieldWall0001` 写入 ACUS，游戏加载时顶掉原版挡板贴图。图片自动规整为 640×480 BC3 格式。

### 修复

- 修复 pgko.dev 谱面下载失败的问题（API 迁移至子域名）。
- 修复外部工具软件内下载卡在 0% 的问题。
- 修复存档编辑器选择收藏品弹窗出现双重标题栏的问题。
- 修复设置页打开卡顿（资源目录改为懒加载）。
- 修复异常弹窗不显示详细错误信息的问题，现在可直接在弹窗中展开完整报错。
- 优化启动速度（约 15%）。
- 其他已知问题修复。

## 打包说明

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\build_windows.ps1" -Version 0.8.0
```

- **懒人包**：`dist\Chuni-Eventer-v0.8.0.zip`
- **Lite 单 exe**：`dist\release\ChuniEventer.exe`

## 升级提示

- 从 v0.7.x 升级：懒人包建议整包替换；Lite 单 exe 可直接覆盖。
- 建议升级前备份当前 ACUS 与自定义资源目录。
