# Chuni-Eventer v0.6.0

## 更新内容

### Stage 功能（本版重点）

- 新增 Stage 管理与编辑主流程（初始化 -> 基础能力 -> 预览能力），补齐从数据读取到配置落盘的闭环。
- 支持为乐曲修改 Stage 关联，便于在既有曲目上快速切换或纠正舞台绑定。
- Stage 相关逻辑进行多轮小修与回滚校正，减少接口调整期间的兼容问题。
- 增加 Stage 预览能力，降低调整参数时反复进游戏验证的成本。

### 自制谱分享 / 上传下载链路

- 新增自制谱分享功能基础框架（客户端侧 init）。
- 补充服务端部署流程，连通分享链路的后端侧支撑。
- 调整上传/下载逻辑，优化交互流程与失败场景处理。
- 针对 SwanClub 相关页面做了 UI 兼容修复，提升展示与操作一致性。

### UI 与交互更新

- 统一表格样式，列表视图在不同页面的视觉和交互表现更一致。
- 修复 Reward 按钮异常，恢复对应入口可用性。

### 资源与格式支持

- 新增 AFB 解析相关能力，为 Stage/资源链路提供格式侧支持。
- 同步补充 Stage 格式分析与文档整理，方便后续维护与扩展。

## 打包说明

- Windows 一键打包：

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\build_windows.ps1" -Version 0.6.0
```

- 详见 `packaging/BUILD_AND_DISTRIBUTION.md`。

## 已知提示

- Stage 与分享链路属于持续迭代模块，若你在旧包上直接覆盖升级，建议先备份现有 ACUS 与缓存目录再验证功能。
