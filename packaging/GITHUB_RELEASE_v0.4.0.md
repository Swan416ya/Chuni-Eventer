# Chuni-Eventer v0.4.0

## 更新内容

- Map / MapArea 编辑页面 UI 优化
  - 地图编辑弹窗整体重构，输入与交互更贴近 Fluent 风格
  - 页签交互优化（页签内新增/删除逻辑、布局与可视反馈改进）
  - 地图格子展示优化（尺寸、缩略图、课题曲标记、信息可读性）
  - MapArea 单项编辑流程简化（减少冗余字段与提示，提升编辑效率）
- 地图资源与预览能力增强
  - ddsMap 预览逻辑优化：更稳健地从资源目录取图
  - dds 预览缓存能力增强，提升地图编辑页加载体验
  - 资源读取过程增加进度反馈
- 本地游戏资源读取能力增强
  - 支持读取并汇总本地游戏资源中的乐曲/舞台等数据
  - 管理页新增“仅查看游戏资源乐曲信息”的入口与筛选能力
- 自制谱下载入口完善
  - 保留并优化 SwanSite 下载自制谱流程
  - 下载/导入链路稳定性与错误提示改进
- 稳定性修复
  - 修复部分线程生命周期导致的崩溃问题
  - 若干对话框交互与加载流程卡顿问题修正

## 重要说明

- 相对 v0.3.0，本次重点是 **Map/MapArea UI 与本地资源读取能力**，以及 **SwanSite 自制谱流程** 的完善。
- `pgko` 相关功能仍在开发验证中，当前版本不作为已完成功能发布。

## 已知问题

- 个别复杂资源场景下仍可能出现预览或导入异常，后续继续修复
- 部分功能在大型资源目录下首次扫描耗时较长
