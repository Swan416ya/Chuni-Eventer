# Chuni-Eventer v0.5.5

## 更新内容

### UI 更新（重点）

- 首页、作品页、新建角色弹窗、地图页完成一轮样式优化，界面层级与信息可读性更清晰。
- pgko 下载弹窗补充来源标注与 UGC 引导：在界面中明确致谢 [Foahh](https://github.com/Foahh)，并提供 UGC -> mgxc 的操作提示与本地缓存入口。
- UGC 帮助区交互优化：文字说明整段直出，演示图改为悬停查看，减少常驻大图对操作区的挤压。

### works 逻辑修复（重点）

- 继续修复 works 分类/索引链路，补齐 works 数据索引，降低导入后作品归类异常、显示不一致的问题。
- 结合前序 `CharaWorks` 相关修复，优化角色-作品关联逻辑，减少角色条目存在但 works 显示缺失的情况。
- 修复角色可能出现的「无解锁」异常表现，提升新导入内容的可用性与一致性。

### 转谱与导入链路（近期修复汇总）

- pgko/mgxc 转谱链路稳定性增强：修复导入完成后的卡死、若干流程阻塞与边界场景卡死问题。
- 封面处理改进：导入时按邻近 UGC 的 `@JACKET` 指定封面重建 DDS，且在压缩前统一规范到 `300x300`。
- 删除乐曲时同步清理 `.cache/dds_preview` 对应缓存，避免删歌后仍显示旧预览图。

### 其它修复与行为调整

- 课题曲选择逻辑优化，降低特定列表条件下的界面卡顿风险。
- 一些逻辑回滚与细节修复，重点收敛在稳定性与可预测行为上。
- 转谱映射细节补充（如颜色映射与链路闭合边界处理）。

## 打包说明

- Windows 一键打包：

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\build_windows.ps1" -Version 0.5.5
```

- 详见 `packaging/BUILD_AND_DISTRIBUTION.md`。
